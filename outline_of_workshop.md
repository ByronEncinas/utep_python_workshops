# Python Workshop for Physicist

A month of student guided problem solving and python discussions, their use and first approach for science undergrads.

## Outline

- Python and Why for Physics?
   - Other Languages to use in Physics (C, C++, Fortran, R, Julia, etc.)
- Basic Syntax
   - Comments, Indentations, do’s and don'ts, convention
   - Datatypes and intrinsic functions
   - Conditionals
   - Loops
   - I/O, Managing Files
- Basic Math with Python (Exercises)
   - Calculating Factorials, Fibonacci Numbers, Evaluating Functions, etc
- Python Library ecosystem
   - Libraries most used by Physicist (Overview with Resources for autodidacts)
      - Neural Networks (Tensorflow, Pytorch)
      - Data Analysis and Visualization (Pandas, Numpy, Matplotlib, yt Project) 
      - Computer Algebra System (SimPy)
   - Using Numpy and Matplotlib
      - Plot Scalar Functions (Temperature)
      - Plot Vector Functions (Electric/Magnetic Fields)
- Project (This idea of a project was not implemented but it is on the original file, so I'll leave it)
   - First week we’ll know how to read files and take inputs
   - Second week we’ll know how to operate with this data
   - Third week we’ll be able to plot this data
   - Fourth week we’ll wrap up with the last details!
