# Python Workshop for Physicist

A month of student guided problem solving and python discussions, their use and first approach for science undergrads.

Instructors:
- Byron Encinas, Graduate Student in Physics, 
- Tadeo Rocha, Undergraduate in Physics
- Also Sanchez, Undergraduate in Computer Science

1. Python and Why for Physics?
   a. Other Languages to use in Physics (C, C++, Fortran, R, Julia, etc.)
2. Basic Syntax
   a. Comments, Indentations, do’s and don'ts, convention
   b. Datatypes and intrinsic functions
   c. Conditionals
   d. Loops
   e. I/O, Managing Files
3. Basic Math with Python (Exercises)
    a. Calculating Factorials, Fibonacci Numbers, Evaluating Functions, etc
4. Python Library ecosystem
   a. Libraries most used by Physicist (Overview with Resources for autodidacts)
       i. Neural Networks (Tensorflow, Pytorch)
       ii. Data Analysis and Visualization (Pandas, Numpy, Matplotlib, yt Project) 
       iii. Computer Algebra System (SimPy)
   b. Using Numpy and Matplotlib
       i. Plot Scalar Functions (Temperature)
       ii. Plot Vector Functions (Electric/Magnetic Fields)
5. Project (This idea of a project was not implemented but it is on the original file, so I'll leave it)
   a. First week we’ll know how to read files and take inputs
   b. Second week we’ll know how to operate with this data
   c. Third week we’ll be able to plot this data
   d. Fourth week we’ll wrap up with the last details!
